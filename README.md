# 🧠 Backend - Student Management Application

This repository contains the server-side logic for the Student Management Application, allowing the teacher full control over student data through APIs.

## 🔗 Project Repositories
- Backend(server): this repository
- [Frontend](https://github.com/eslam-cmd/e-school-client): contains the design code

## 🛠️ Technologies Used

- Node.js
- Express
- PostgreSQL
- neon-tech
- REST API

## 📦 Core Features

- Create student accounts and generate a unique ID for each student
- Save and update exam and quiz grades
- Record daily attendance
- Verify student identity through the ID and return their data

## 🚀 How to Run

```bash
npm install
npm run dev
